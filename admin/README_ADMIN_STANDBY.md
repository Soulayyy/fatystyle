# Admin Faty Style en stand-by

L’administration n’est pas développée dans cette livraison. Le site reste volontairement statique, autonome et sans WordPress.

La structure est déjà préparée pour une phase suivante :

- textes du site ;
- coordonnées ;
- hero ;
- prestations ;
- catégories de créations ;
- photos et galerie ;
- contact ;
- réseaux sociaux Facebook et Instagram.
- SEO local : titres, descriptions, mots-clés, Open Graph et données structurées.

Le futur admin devra utiliser `data/content.json` comme source de contenu et les dossiers `assets/images/` déjà structurés par rubrique.

Aucun bouton Admin, identifiant temporaire ou faux back-office n’est actif sur le site public.
